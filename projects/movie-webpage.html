Movie Webpage Project

Context

In this project, me and my partner Crystopher were tasked with creating a website about a fake movie. We decided to create a fake horror movie which was a sequel to a comically terrible movie, naming it "Sequential: The Sequel"

Content

To start off our project we created a plan which required both CSS and HTML elements.

HTML elements

  • Headers
  • Paragraphs
  • Images (Local)
  • Divs and Spans
  • Fonts
  • Textboxes
  • Submit Buttons

CSS elements

  • Coloring
  • Background
  • Centering
  • Element, Class, and ID selectors

Reflection

Challenges

In this project, we experienced few problems. But a problem we did experience was figuring out how to center the images, it took us a while to realize that for text-align:center to work we needed to place the image inside of a div, which ended up working out to fix our problem.

Takeaways

Some key takeaways from this project is me figuring out how to use the background element with a linear gradient, which was a very good thing to learn. As well as how to make a button to help submit feedback from people into our website, even if at the moment it is not functional.

Next Steps

Some steps I intend to take after this project is to take what I learned from this project and put it into other projects I will make in the future.
